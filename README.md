# Trabajo Integrador - Virtualización

Este repositorio contiene el desarrollo del Trabajo Integrador, compuesto por el informe técnico en formato PDF y el script en Python que automatiza el cálculo del promedio aritmético a partir de 5 números ingresados por consola.

## Estructura del Proyecto

La organización de los archivos en el directorio es la siguiente:

```text
.
├── TP_Integrador_ArqYSis.pdf    # Documentación e informe técnico del TP. Dentro del mismo se incluyen las capturas de pantalla.
└── main.py                      # Script principal con el código de la calculadora
```

## Requisitos

Para ejecutar el programa es necesario tener instalado Python.

Puede verificarse la instalación ejecutando:

```bash
python --version
```

## Ejecución

1. Abrir una terminal.
2. Navegar hasta el directorio donde se encuentra el proyecto.
3. Ejecutar el siguiente comando:

```bash
python main.py
```

## Funcionamiento

El programa solicita al usuario el ingreso de 5 números por consola, calcula el promedio aritmético y muestra el resultado en pantalla.

### Ejemplo de ejecución

```text
Ingrese el 1 número: 10
Ingrese el 2 número: 20
Ingrese el 3 número: 30
Ingrese el 4 número: 40
Ingrese el 5 número: 50

El promedio de los 5 números ingresados es: 30.0
```
