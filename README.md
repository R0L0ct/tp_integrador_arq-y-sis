# Trabajo Integrador - Virtualizacion

Este repositorio contiene el desarrollo del Trabajo Integrador, compuesto por el informe técnico en formato PDF y el script en Python que automatiza el cálculo del promedio aritmético a partir de 5 números ingresados por consola.

## 📋 Estructura del Proyecto

La organización de los archivos en el directorio es la siguiente:

```text
.
├── PDF tp integrador    # Documentación e informe técnico del TP. Dentro del mismo se incluyen las capturas de pantalla.
└── main.py              # Script principal con el código de la calculadora
